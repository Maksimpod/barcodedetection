# О программе
Данная программа предназначена для распознавания и декодирования линейных штрихкодов EAN-8 и EAN-13. Функциональная часть реализована с помощью библиотеки OpenCV, для интерфейса использовалась библиотека FreeSimpleGUI.

# Руководство
Требуется установить Python версии 3.7 и выше, систему управления пакетами pip, а также библиотеки OpenCV, NumPy и FreeSimpleGUI. При тестировании использовались OpenCV 4.12.0.88, NumPy 2.2.6 и FreeSimpleGUI 5.2.0.
1. Запустить файл командой ```python BarcodeDetection.py ```
2. В появившемся окне инерфейса в верхней части с помощью кнопки "Обзор" выбрать файл изображения или ввести в строке слева путь к файлу.
3. Нажать кнопку "Начать". После этого появится отдельное окно с исходным изображением, на котором будет выделена область штрихкода и декодированное значение. Декодированное значение также выведится в виде текста в области "Вывод" программы.
